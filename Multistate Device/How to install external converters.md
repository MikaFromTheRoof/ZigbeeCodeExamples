* make sure, that you are using the new windfront frontend of Zigbee2MQTT
* select "Settings" on the sidebar
* select "Dev console" on the topbar
* select "External converters"
* below "Select converter to edit" choose "N/A - Create new converter" from the dropdown menu
* you can choose a name for the converter freely
* below "code" paste the code from the external converter js-file provided
* hit "save"
* pair or repair the device you added the external converter for
* the device should now be diplayed as "Supported: external"
* maybe reconfigure and re-interview the device by hitting the corresponding buttons in the device list, if changes you did to the device or to the converter don't seem to appear correctly in Zigbee2MQTT
